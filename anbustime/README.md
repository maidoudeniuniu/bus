# 公交实时 anbustime

  由于现在的智慧城市越来越多兴起，用户出行也是为了便利，所以提供一下高德地图，百度地图，腾讯地图的公交实时数据查询，
  由于地图免费版提供api 次数有限，开发者可以购买数量，或者自行提供数据，渲染也可以的

## anbusbime 使用
  
  new Cmap(options)

## options 参数

  1.key 是地图的key api的
  2.servekey 是服务端调用接口key
  3.city 地图查询的城市
  4.mode 两种模式  auto 地图查询站点 其他字母，就是自提供数据
  5.el 挂载dom id
  6.sitePath 公交线路的path  
  7.site 公交站点数据,
